# 大鸿小说创作超级工作台

> 从一句话创意到500万字长篇网文的全流程方法论。
> AI不是写手，AI是小说团队——模拟真实网文工作室架构，10个专家角色分工协作，由总策划CEO统一对接作者。

## 核心理念

**创作铁律：70%剧情 + 30%干货**
- 剧情是骨架，干货是血肉
- 干货来自知识库，自然嵌入不堆砌
- 每章2-3个干货点，每个不超过150字
- 人物对白是干货的最佳载体
- 禁止脱离剧情独立说教

## 10人专家团队架构

| # | 角色 | 职责 | 输出物 |
|---|------|------|--------|
| 1 | 总策划（CEO） | 跟作者沟通，统筹全局，调度团队 | 市场分析、核心卖点、对标作品 |
| 2 | 世界观设计师 | 搭建世界、力量体系、行业规则 | `世界设定.md` `力量体系.md` `知识库.md` |
| 3 | 剧情架构师 | 百章大纲→三百章大纲→百万字路线图 | `大纲.md` `爆点路线图.md` `伏笔规划.md` |
| 4 | 角色设计师 | 主角+配角+反派全面塑造 | `人物设定.md` `人物成长线.md` |
| 5 | 正文写手 | 按细纲逐章创作，保持人设和节奏 | `第XXX章.md` |
| 6 | 番茄编辑 | 黄金三章、钩子、爽点、情绪价值审核 | `编辑审核报告` |
| 7 | 知识顾问 | 中医/风水/健身/商业/历史等专业审核 | `知识审核报告` |
| 8 | 数据运营 | 模拟平台数据：完读率/追读率/弃书风险 | `数据预测报告` |
| 9 | 创意孵化器 | 一句话→5个版本方向 | 多版本创意方案 |
| 10 | 爆款评分器 | 100分制评分 | 评分卡 + A/B/C/D评级 |

## 完整工作流（13步流水线）

```
一句话创意
  ↓
① 创意孵化器 → 生成5个版本方向
② 爆款评分器 → 100分制评分
③ 总策划CEO → 市场分析/核心卖点
④ 世界观设计师 → 世界设定/力量体系
⑤ 角色设计师 → 人物设定/成长线
⑥ 剧情架构师 → 大纲/爆点路线图
⑦ 知识顾问 → 专业领域知识审核
⑧ 正文写手 → 逐章输出（每章3000-5000字）
⑨ 番茄编辑 → 爽点/钩子/情绪审核
⑩ 数据运营 → 完读率/追读率预测
⑪ 持续优化循环 → 调整后续章节
⑫ 进度追踪 → 更新 progress.json
⑬ 下一章 → 回到⑧
```

## 防崩机制

- **剧情防崩**：衔接检查、大纲校验、伏笔管理、节奏控制、爽点密度
- **人物防崩**：对话风格锁定、禁止OOC、禁止战力崩坏
- **知识防崩**：专业知识必须来自知识库，不得编造

## 黄金三章标准

| 章节 | 核心任务 | 技术要求 |
|------|---------|---------|
| 第一章 | 亮金手指 + 核心冲突 | 开头300字内必须出现金手指或生死危机 |
| 第二章 | 首次破局 + 世界观扩展 | 展示主角专业能力，引入第一个客户/对手 |
| 第三章 | 揭秘更大阴谋 + 引出关键人物 | 信息密度高但递进有序，每2000字一个转折 |

## 伏笔管理系统

| 级别 | 回收周期 | 说明 |
|------|---------|------|
| 小伏笔 | 3-5章 | 章节内或近章回收 |
| 中伏笔 | 一卷内 | 当卷结束前回收 |
| 大伏笔 | 跨卷 | 跨越多卷回收 |
| 终极伏笔 | 全书 | 结局才揭示 |

## 快速启动

1. **一句话创意** — 写下核心创意
2. **创意孵化** — 让孵化器生成5个版本方向
3. **爆款评分** — 对选定方向进行100分制评分
4. **市场分析** — 调研目标平台的赛道和对标作品
5. **建知识库** — 上传PDF资料，提取文本，建立金句库
6. **写大纲** — 确定卷数、每卷主题、身份升级路线
7. **设人物** — 角色设定和对话风格锁定
8. **建progress.json** — 初始化进度追踪系统
9. **写黄金三章** — 第一章300字亮金手指
10. **启动自动化** — 配置定时任务，持续产出
